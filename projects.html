Projects

I've completed a variety of projects for both academic purposes, to teach myself, or for fun. This page showcases some of my favourite projects, along with links to their GitHub repos to view their source code or links to download them for your own use!

MeFit Competition Management Web App

This web application allows corporate partners to create and manage athletic competitions, as well as athletes participating in them, and their results. This application was created for my Databases I (CSI2532) class at uOttawa, and is developed using HTML5, CSS, and PHP, in addition to a PostgreSQL database.

Lights Out! Mobile Game

This is an Android game developed in Java using Android Studio, in which the player has a grid with the simple task of turning off all the lights. The catch? Toggling a light toggles the adjacent ones as well. This game is based on previous work I had done in an Intro to Programming class.

Clinic Management Application

This is an Android application developed in Java using Android Studio that allows patients and staff to conduct their affairs at a clinic. Staff can manage their hours of operation or the services they offer, while patients can search these services and book appointments with doctors. The program uses a Firebase database to store its data and was created for my Introduction to Software Engineering (SEG2505) class at uOttawa.

Tic Tac Toe

Implementations of the classic Tic Tac Toe game written in a variety of languages and a variety of tools, generally to help me learn a language and associated skills (such as basic UI design in that language, as well as the proper use of arrays or matrices). Implemented are an Android version in Java, two C# versions in .NET and Unity, as well as console-based Python, Java, Prolog, and Scheme versions.

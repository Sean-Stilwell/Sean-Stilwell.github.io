MeFit Competition Management Web Application

This web application allows corporate partners to create and manage athletic competitions, as well as athletes participating in them, and their results. This application was created for my Databases I (CSI2532) class at uOttawa, and is developed using HTML5, CSS, and PHP, in addition to a PostgreSQL database.

Lights Out! Mobile Game

This is an Android game developed in Java using Android Studio, in which the player has a grid with the simple task of turning off all the lights. The catch? Toggling a light toggles the adjacent ones as well. This game is based on previous work I had done in an Intro to Programming class.

Tic-Tac-Toe

GitHub Repo

When learning a new language, I often try to code a simple Tic Tac Toe game, as it teaches a variety of skills (lists, conditionality, simple UI design). All of my implementations can be found at the repository above, along with brief explanations below:

Unity Implementation -

This version was created on Unity Pro and is coded in C#. I created this to teach myself how to use Unity for 2D game design, as well as to perfect the C# version I created in my .NET implementation.

.NET Framework Implementation -

This version was created on Visual Studio 2019 using standard Windows Form Applications from the .NET Framework, it is coded in C#. This version uses a MVC methodology, with a Board class modelling the Tic Tac Toe game, a visual display of the model, and a controller class to run the game.

Python Implementation -

This is a text-based version of Tic Tac Toe, coded in Python. It was originally created as part of my Intro to Computing I class for Professor Diana Inkpen. This served as the basis for the Java version.

Java Implementation -

This is a text-based version of Tic Tac Toe, coded in Java. It was originally created prior to my Intro to Computing II class, so I could learn basic Java before starting. This implementation was also used as an outline for the C#/.NET implementation.

Racket/Scheme Implementation -

This is another simple text-based version of Tic Tac Toe, albeit very different from the Python or Java implementations. This uses the functional programming paradigm and is coded in Racket/Scheme, which heavily uses recursion.

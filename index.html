Projects!

I've completed many projects for both academic and personal reasons. Below is a showcase of some of my work, along with links to the GitHub repos for personal projects.

MeFit Competition Management Web Application (Video Demo)

This web application allows corporate partners to manage their competitions quickly and effectively. Partners can create competitions, which contain a series of events, and then add athletes and their results to the system. A PostgreSQL database holds all data, facilitated by PHP to display and retrieve information.

This project was completed as part of my Databases I class and received a mark of 97%.

Clinic Management Android Application (French Documentation)

This mobile application allows administrators, employees, and patients at clinics to conduct all interactions with the clinic. Employees can set their hours and services provided, while patients can make appointments and search available doctors. A Firebase database is used for the data, and the program is written in Java.

This group project was completed for an Introduction to Software Engineering class and received a mark of 91%.

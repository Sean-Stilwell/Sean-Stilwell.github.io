uOttawa Clinic Android Application

This mobile application allows administrators, employees, and patients at the uOttawa Clinic to conduct all interactions with the clinic. (Note: this app was made for a Software Engineering class and is not the official application)

This application was developed in a group for Android using Android Studio, and was coded in Java. The database uses Firebase, the built-in database system recommended for Android Studio. This project was performed using an Agile methodology, with the final project consisting of four distinct deliverables.

Features:

  • Employees and Patients can create secure accounts with encryption protecting their passwords.
  • Employees can manage the services they offer, as well as their availability times each day.
  • Patients can search for services based on availability or popularity.
  • Patients can make appointments with employees and then evaluate their experience.

Each deliverable brought new functionality to the app. The first brought account creation and logging in, the second brought administrator functionality, the third brought employee functionality, and the final brought patient functionality. We created UML models to plan our work, developed them, and created unit test to ensure proper functionality.
